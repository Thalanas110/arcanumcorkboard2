# AnonCorkboardThree

This repository contains a Vite + React + TypeScript project scaffold with shadcn-ui and Tailwind.

## How to edit and run locally

1. Clone the repository:

```powershell
git clone https://github.com/Thalanas110/arcanumcorkboard2.git
cd arcanumcorkboard2
```

2. Install dependencies and start the dev server:

```powershell
npm install
npm run dev
```

3. Edit files in `src/` with your preferred editor (VS Code recommended).

## Technologies

- Vite
- TypeScript
- React
- Tailwind CSS
- shadcn-ui

## Notes

- If this project contains a `package-lock.json` and `node_modules/`, regenerate the lockfile after removing or changing dependencies by running `npm install`.
- To fully remove any installed packages locally, run `rm -rf node_modules package-lock.json` (or the Windows equivalent) and then `npm install`.
